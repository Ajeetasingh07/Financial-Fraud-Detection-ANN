# Financial Fraud Detection using ANN

## Project Overview
This project aims to detect fraudulent credit card transactions using an Artificial Neural Network (ANN). Since the dataset is highly imbalanced, the SMOTE technique will be used to balance the classes before training the model.

## Dataset
- Credit Card Fraud Detection Dataset (Kaggle)

## Tools and Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- TensorFlow/Keras
- SMOTE (imbalanced-learn)
